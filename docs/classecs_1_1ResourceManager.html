std::shared_ptr< AUserComponent > ecs::ResourceManager::LoadUserComponent ( const std::string &  resourceID)

Returns an instance of the AUserComponent child class contained in the shared library with the name given in parameter. The library should contain a method named create, which will return a pointer to the instance of the class. This method can be generated easily using the MANAGED_RESOURCE macro in metadata.hpp.

Parameters
